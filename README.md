# 🎯 Redrob AI Challenge: Candidate Discovery & Ranking

Production-grade ranking pipeline built to discover top-tier applied ML engineering talent while systematically dodging dataset traps.

## 🪤 Strategic Trap Handling
- **Keyword Stuffing Mitigation:** Drops non-technical background roles (Marketing/Ops) listing AI tags.
- **IT Consulting Bias Control:** Adjusts score weights down for mass IT service firms, favoring verified product environment backgrounds.
- **Liveness Optimization:** Applies a penalty to historical "ghost" profiles inactive for over 180 days.
